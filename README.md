# Henna Studio Management & Booking System

## Project Overview
This project is being developed for the Database Management System (DBMS) course.

The system will help manage a henna studio by allowing customers to view designs, book appointments, make payments, and submit reviews. The admin can manage bookings, customers, designs, and payment records.

## Main Features
- Customer Registration
- Customer Login
- Henna Design Gallery
- Appointment Booking
- Booking Management
- Payment Management
- Customer Reviews

## Proposed Database Tables
1. Customers
2. Artists
3. Designs
4. Bookings
5. Payments
6. Reviews
